	SET-UP
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
acces en ligne de notre app  : https://app-titus.herokuapp.com

Active la migration et aussi le seed :

$ rails db:create
$ rails db:migrate

notre github  

$ git clone https://github.com/titsu11/Eventbrite

Go to the folder and run $ bundle install

	Pour lancer
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Lance ton serveur  :

$ rails server
